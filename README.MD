## 使用到的设计模式

1. 构造函数模式
2. 工厂模式: 商品创建,\$('xxx')
3. 单例模式: 购物车
4. 装饰器模式: 日志打点
5. 代理模式: 折扣处理,proxy
6. 观察者模式: 网页事件,promise
7. 状态模式: 商品按钮(添加,移出购物车)
8. 模版方法模式: 渲染有统一模式(render),包含了各子模块的渲染
